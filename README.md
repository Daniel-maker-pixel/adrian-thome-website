## Struktur

```

├── index.html          # Hauptseite
├── impressum.html      # Impressum
├── datenschutz.html    # Datenschutzerklärung
├── netlify.toml        # Netlify-Konfiguration
└── ANLEITUNG.md        # Deployment-Anleitung
```

## Deployment

Automatisch via Netlify bei jedem Push auf `main`.

## Änderungen vornehmen

1. Datei direkt auf GitHub bearbeiten (Stift-Symbol)
2. Änderung speichern ("Commit changes")
3. Netlify deployed automatisch — live in ~30 Sekunden

## Platzhalter

Folgende Werte noch eintragen:
- `KAJABI_BUCHUNG_URL` → Kajabi Buchungsseiten-URL
- `DEINE_TELEFONNUMMER` → eigene Telefonnummer  
- `DEINE-DOMAIN.de` → eigene Domain
